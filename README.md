# Keypad
Directional arrow keys, 4 Macro keys, 1 Knob, Display

## Part Selection
### MCU
- RP2040: Raspberry Pi Pico
### Input Devices
- EC-11 Rotary Encoder (with push button)
### Display
- I2C OLED Display: SSD1306
### Connectivity
- USB-C Breakout Board
- ESD Protection Diode: TPD2E001
